# 트릭컬 최애 사도 표 메이커
성격별/종족별 최애 사도를 골라서 이미지로 저장하고 공유할 수 있다!!

*우로스는 모든 성격에서 선택할 수 있습니다.

사이트 링크: https://trickal-favorite-apostle.vercel.app/

## 기술 스택
- Framework: React (Vite)

- Libraries: html2canvas, styled-components

- Deployment: Vercel


### 저작권 관련
- 본 프로젝트는 '**에피드게임즈(EPIDGames)**' 의 게임 '트릭컬 리바이브'의 팬 메이드 2차 창작물입니다.

- 모든 사도 아이콘 및 게임 리소스에 대한 저작권과 권한은 에피드게임즈에 있습니다.

- 본 서비스는 비영리 목적으로 운영됩니다.
